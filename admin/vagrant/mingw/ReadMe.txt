description:
these scripts make cross-compilation using mingw64 in a fedora64 vm and release WIN32 exe packages of percolator and converters.

Prerequisites:
* vagrant 1.2.2
* VirtualBox
* git
* openssh

Instructions:
1. in build_win.sh set release value to the desired path and branch value to the desired 
branch.
3. $ bash build_win.sh
